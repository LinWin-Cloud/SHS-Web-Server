LinwinSoft SHS


This is a default page of the SHS server. You can put the html,css,js file on '/var/web/www/'

Configure


1. Edit /usr/SHS/config/http_service.json

The content in the file: 

        {
            "port": "8888",
            "IP": "127.0.0.1",
            "default": "index.html,index.htm",
            "html": "/home/ubuntu-linux/start-http-server/default",
            "charset": "utf-8",
            "error_page": "../default/error/",
            "Access-Control-Allow-Origin": "",
            "Access-Control-Allow-Credentials": "false"
            "Access-Control-Allow-Methods": "get,post",
            "php_requests" : "1000",
            "requests": "1000000"
        }

2. Input Command in the Terminal: shs start Then will start SHS.
